Installation:
------------

    git clone https://github.com/radome/vim-conf.git ~/.vim

Create symlink:
--------------

    ln -s ~/.vim/vimrc ~/.vimrc

Install plugins:
----------------
Start Vim then run ```:BundleInstall``` to fresh install all the plugins.

Update plugins:
----------------
Start Vim then run ```:BundleInstall!``` to update all the plugins.
